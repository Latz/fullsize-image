# Privacy Policy for Fullsize Image

**Last Updated:** January 19, 2026

## Overview

The Fullsize Image browser extension ("Extension") automatically resizes images to fit your browser's viewport. This privacy policy explains what information the Extension collects and how it is used.

## Data Collection

### Information the Extension Does NOT Collect
The Fullsize Image Extension **does not** collect, store, or transmit any personal information, including:
- No browsing history
- No website data or content
- No personal identifiers
- No cookies or tracking data
- No user preferences or settings (beyond extension state)

### Information the Extension DOES Use
The Extension only processes data locally in your browser:
- **Extension State:** Stores a simple "active" or "sleeping" state in your browser's local storage to remember whether the extension is enabled or disabled
- **Image URLs:** Temporarily checks image URLs to determine if they are image files (jpg, jpeg, png, gif) when determining whether to apply resizing

## How the Extension Works

1. **Local Processing Only:** All image resizing operations happen entirely within your browser. No data is sent to external servers.
2. **No Network Requests:** The Extension does not make any network requests or communicate with any external services.
3. **No Data Storage:** The Extension does not store any user data on external servers.

## Permissions Explained

The Extension requires the following permissions to function:

- **activeTab:** To access and modify images on the current webpage
- **scripting:** To inject JavaScript that resizes images
- **storage:** To save the extension's active/sleeping state
- **host_permissions:** To access images on web pages you visit

## Third-Party Services

The Extension does not use any third-party services, analytics, or tracking mechanisms.

## Data Security

Since all operations are performed locally in your browser and no data is transmitted externally, there are no security concerns related to data transmission or storage on remote servers.

## Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted on the Chrome Web Store listing and in the extension's repository.
